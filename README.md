Giao diện chương trình khi các phím Num, Caps, Scroll đang bật:
![GUI](Out/Img/GUI.png?raw=true)

Giao diện khi chỉ có NumLock đang bật:
![GUI](Out/Img/OnlyNumLock.png?raw=true)
